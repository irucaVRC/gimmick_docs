# アバターシンクロ&ポーズシステム 概要
本ギミックは、ワールドに設置することでプレイヤーの動きをワールドに設置したアバターに鏡合わせでシンクロさせることができます。プレイヤーのアバターとワールドに設置したアバターが異なるアバターでもできる限り同じような姿勢になるようにしています(すべてのアバターでの動作を保証するものではありません)。

また、シンクロを止め、手や足などをつかんで任意の姿勢にさせることができます。
掴める場所は、頭・腰・両手・両足です。シンクロを止めた場合、アバターの掴める場所に白い球体が表示されます。これを掴むことでアバターを自由な姿勢にできます。この球体は、カメラには写りません。

すべてローカル動作です。

複数のアバターを設置し、切り替えることができます。

以下の用途などに使えそうだと想定しています。
- アバター・衣装の展示用ディスプレイとして
    - サンプルアバター等での直接的な試着を行わなくても、プレイヤーと同じ動きをするため、購入検討中のユーザーへ「着用時のシルエット」や「動いた時の見え方」を効果的に伝えることができます。
- ソロでのツーショット撮影に
    - 自分の隣にアバターを配置することで、一人でもペア写真や、複数人でいるような構図の撮影が楽しめます。お気に入りのアバター同士を並べた撮影にも最適です。
- 添い寝
    - 自由にポーズをつけて、あなただけの等身大バーチャル抱き枕に。腕枕をしてもらったり、背中合わせで寝転んだり、睡眠ワールドでの一人時間に最高の安心感を提供します。
